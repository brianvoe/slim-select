## Features
- No Dependencies
- 20kb - 5kb gzip
- Single Select
- Multi Select
- User Addable Options
- Html Options
- Settable Data
- Callback Events
- Placeholders
- Search
- Disable Options
- Light CSS
- Light Color Scheme
- Style and Class Inheritance
- Clean Animations
- Performant

## Installation
```bash
npm install slim-select
```

### or

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/slim-select/<version>/slimselect.min.js"></script>
<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/slim-select/<version>/slimselect.min.css"> 
```

## Usage
```javascript
import SlimSelect from 'slim-select'

new SlimSelect({
  select: '#slim-select'
})
```
