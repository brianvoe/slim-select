## Installation
```bash
npm install slim-select
```

### or

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/slim-select/<version>/slimselect.min.js"></script>
<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/slim-select/<version>/slimselect.min.css"> 
```

## Usage
```javascript
import SlimSelect from 'slim-select'

new SlimSelect({
  select: '#slim-select'
})
```

## Capybara
Assumption is that you have a label who's parent also contains the select list you are choosing from.

```ruby
def js_select(item_text, options)
  container = find(:xpath, "//parent::*[label[text()='#{options[:from]}']]")

  within "##{container[:id]}", visible: false do
    find('.ss-arrow').click
    input = find(".ss-search input").native
    input.send_keys(item_text)
    find('div.ss-list').click
  end
end
```
